# Nexus Rentals

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Neon PostgreSQL
- Stripe Connect
- Vercel

## Development Rules

- Preserve current UI styling unless explicitly requested.
- Do not modify authentication without approval.
- Do not modify database schema without explaining the plan.
- Do not modify Stripe payment logic without explaining the plan.
- Prefer small safe changes.
- Explain implementation plans before coding.

## Current Priorities

1. Onboard first real landlords
2. Stripe live payments
3. Tenant applications
4. Admin analytics
5. Production stability
